1. Category and Subcategory

This application groups account records into subcategories, and group subcategories into categores. After click in the upper-right corner, then click "Import built-in category list" , a set of categories and subcategories will be generated. You can add/delete/modify these categories and subcategories for your requirement in "Category" tab.

Notice:

  • When adding subcategory, the subcategory can be created as an incoming subcategory which only contains incoming records, or an expense subcategory which only contains expense records.
  • To edit/delete a category/subcategory, long press the name of the category, select 'Edit' or 'Delete' in the popup menu.

2. Expense/incoming tracking

Enter 'Category' tab, click one subcategory or enter 'Records' tab, then click "+Expense" or "+Incoming" button, fill information of the new record, click "Add" in the upper right corner.

FieldsNotes
Amount You can input +,-,* to do simple calulation in amount input field.
Merchant You can select from merchants previously recorded in same subcategory by clicking button next to merchant input field.
Label Mark record with a label defined in "Settings"/"Label Management" page. The summary of different labels can be viewed at "Chart" tab.
Payment Select a payment method defined in "Settings/Payment Management" page. The record will use the payment's currency. The summary of different payments can be viewed at "Chart" tab.
Date Click right arrow button, fill information in expanded view.
Detail Click "Detail" tab and fill the information。
Image Click "Image" tab, you can attach a image with the record(For example, add an image of receipt or invoice). You can also add an image by dragging it from other app to the Image tab or "+ Expense/Incoming" button.

You can use your finger to drag a green rectangle to select a region of the image

Following action can be done by clicking the buttons in the bottom of the tab.

  • Photos: add image from Photos app
  • Camera: add image by taking a picture using camera
  • Crop: crop the selected region out of image
  • Analyse: list of money amount, date, merchants will be prompt in a new window if there is any contained in selected image region(or the whole image if no selection). Three categories will also be prompt for selection.
  • Clear: Remove the image
Continuous add The dialog will not be closed after clicking "Add",so that you can add next record. This is also supported in label, category and periodic record template adding dailog。

Notice：

  • The merchants information can only be recongnized when there is at least one previous record contains the information.
  • It is better to select relavent region before click "Analyse" to improve accuracy
  • The suggested categories in “Analyse” is based on existing record's details, merchants and text in attached images. If there are too few of these information, the suggestion may not be accurate.

3. Records

Enter 'Category' tab, click one subcategory or enter 'Records' tab, here you can view all the records in a particular day. The total expense and income of the whole year and month will also be shown. Click the year, month and day text on the top will pop up menu to switch displaying date.

Notice:

  • The popup menu for switching date will only contain date which has records.
  • You can select "All" to show records of the whole year or month.
  • You can click the record to view more information, you can view the attached image in image tab.
  • You can edit/delete record by long pressing(mouse right click in MacOS) the record.
  • You can click "Edit" button in the upper-right corner then select some records and click "Move" to move records to other subcategory, or click "Label" to label them (reference Label Management in section 5) or click "Delete" to delete them. Click "Done" button in the upper-right corner to finish edit state.
  • In "Category" tab, click "Data Recycle". Here, you can view records deleted but not synchronized with iCloud. To restore the records, select records which need to be restored and click "Restore" or click "Restore and Move" if the record's subcategory does not exists.
  • In "Category" tab, click "Search records". Here, you can search records which detail, merchant or text in attached image contains certain keyword. You can move, lable or delete records by clicking "Edit" button in the upper-right corner and click corresponding button in the bottom.

4. Chart

In "Chart" tab, you can view statistic on the records by different angle. For example, show total sum of the records for each year, then in each year, show total sum of the records for each month. You can change the summary fields by click Edit in "summary method and order" section. Dragging in the right of the summary method to change the order. Following summary methods are supported:
  • Summary By Label
  • Summary By Payment
  • Yearly Summary
  • Monthly Summary
  • Daily Summary
  • Summary By Category
  • Summary By Subcategory
  • Summary By Merchant
After entering a summary page, click in the upper right corner to return back to home summary page. Click to add this page to quick access pages. Added quick access links will be listed under home summary page. When clicking the quick access link, related page will be accessed, click will remove the quick access link. If the access path of current summarize page contains year\month\day or year\month or year only and these fields's value are same with current date. This app will prompt "Do you want to keep the date on access path always uptodate?" when adding this page to quick access. If selecting 'Yes', the date of the quick access link will always be current date. For example, if current date is 2021/7/1, when adding summarize page of Food category in 2021/7 to quick access. When entering Augest of 2021, the link will point to 2021/8's summarize page.

5. Settings

Enter the “Settings” tab, here you can use following functions and settings.

Upgrade App
(unrestricted use) 		Click"Upgrade App at price XXX" to pay and upgrade the app. After upgrade, text “Purchased” will replace the upgrade button.

Notice:
If you reinstalled the app, or installed the app on another device after upgrading, you can click "Restore Purchase" to restore purchase state.
Sync iCloud To avoid data lost, click this button to sync categories, subcategories and records to iCloud, or synchronize data between different devices(each device must login to same icloud account). The local updates will be uploaded to iCloud, and the remote changes will be downloaded to local device. Note that if you have updates on same field of the same category or subcategory or record from both local and iCloud, the local update will overwrite iCloud update.
Auto-Sync Automatically synchronize iCloud every 2 minutes.
Owner authenticate Authenticate using FaceID, TouchID or Password after entering the app to protected your privacy.
Export to file Export categories, subcategories and records data to CSV file. Note that image data in records will be ignored.
Import from file Import categories, subcategories and records data from CSV file. Note that only data in CSV file that does not exists in local device will be imported.
Auto-insert template You can config the app to add record automatically in a specific day of each month (For example, add X dollar expense at day 31 of each month for iCloud monthly service fee). If a date is not valid in certain month, the auto-created record will be set for the last day of the month. (February does not have day 31, so the new record date will be set to 29 February or 28 February)
Label Management Add self defined label, you can mark each record with one of labels you defined. The summary of different labels can be viewed at "Chart" tab.
Payment Management Add payment, you need to define a payment if you want to add record with non-default currency. The summary of different payments can be viewed at "Chart" tab.
Question and Feedback If you have any question or suggestion about the app, you can click "Questions". If notification privilege for this app is not enabled, app will display a dialog when enter the feedback page. Click 'Yes' to enable to the privilege to receive reply notification.

6. Import/Export cvs file format

Each line in imported/exported cvs file contains data of one category, subcategory, label or record, seperated by comma. If there is a comma or change line in the field, use \, (full-width comma) and \n.

Category contains following fields:
  1. 0
  2. Category id, any string that can uniquely identify the category.
  3. Category name
Subcategory contains following fields:
  1. 1
  2. Subcategory id, any string that can uniquely identify the subcategory.
  3. Subcategory name
  4. Parent category id
  5. true -- incoming subcategory, false -- Expense subcategory
Record contains following fields:
  1. 2
  2. Record id, any string that can uniquely identify the record.
  3. Parent subcategory id
  4. Eecord date(year-month-day hour:minute), ex. 2022-01-12 05:15
  5. Amount, ex. 100.00
  6. 0
  7. Merchant
  8. Detail
  9. Label id
Label contains following fields:
  1. 3
  2. Label id，any string that can uniquely identify the label.
  3. Label name
  4. Label color：RGB color value ranged from 0 to 1, seperated by |， ex. "0.5|0|0.5"
Payment contains following fields:
  1. 4
  2. Payment id，any string that can uniquely identify the payment.
  3. Payment name
  4. Payment currency：ex. "USD"
